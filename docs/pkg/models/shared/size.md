# Size

The size of the generated images. Must be one of `256x256`, `512x512`, or `1024x1024`.


## Values

| Name                                | Value                               |
| ----------------------------------- | ----------------------------------- |
| `SizeTwoHundredAndFiftySixx256`     | 256x256                             |
| `SizeFiveHundredAndTwelvex512`      | 512x512                             |
| `SizeOneThousandAndTwentyFourx1024` | 1024x1024                           |
