# Voice

The voice to use when generating the audio. Supported voices are `alloy`, `echo`, `fable`, `onyx`, `nova`, and `shimmer`.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `VoiceAlloy`   | alloy          |
| `VoiceEcho`    | echo           |
| `VoiceFable`   | fable          |
| `VoiceOnyx`    | onyx           |
| `VoiceNova`    | nova           |
| `VoiceShimmer` | shimmer        |
